# Automation_Projects

**Window_Forensics_Project - Analyzer:** Automatic Analysis project named **Analyzer**, working with memory and HDD files to extract sensitive information and present the findings to the researcher. 


**Network_Security - Domain Mapper:** Thoroughly check networks for weakspots and defend against attacks.
A script transforming a Linux system into a Honeypot to detect and alert for anomalies. Users can select the services to start and monitor, with activity data logged for auditing. The script also uses log details to conduct in-depth scans, collecting information such as the origin country, organization, contact information, open ports, and services of each user attempting access. While active, the Alerter provides a live-mode view of the Honeypot's ongoing activity, facilitating real-time monitoring
