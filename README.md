# Automation_Projects

**Window_Forensics_Project - Analyzer:** A bash base automatic analysis project named **Analyzer**, working with memory and HDD files to extract sensitive information and present the findings to the researcher. 


**Network_Security - Domain Mapper:** A bash based network reconnaissance and Active Directory assessment tool that automates scanning, enumeration, and basic exploitation workflows in a Linux environment.


**Network_Researcher_Project - NipeRecon:**A bash based tool that automates anonymized SSH reconnaissance by preparing the environment, routing traffic through Nipe, and collecting structured network and system data using Nmap and Whois.
